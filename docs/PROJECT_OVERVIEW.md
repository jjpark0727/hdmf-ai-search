# AI서치 프로젝트 개요

## 시스템 목적

### 비즈니스 배경
- **문제**: 당사 임직원들이 사내외 정보를 검색 및 정리하는데 많은 시간 소요
- **현황**: 문서별로 검색하거나 수작업으로 비교 분석
- **목표**: 생성형 AI 및 Agent 기반 서비스로 업무 효율성 향상

### 단계별 개발 계획
- **1단계** (2026.10월 오픈): 사용자가 업로드한 문서를 기반으로 질의응답, 요약, 번역, 당사 양식에 맞는 보고서 생성 기능 제공
- **2단계** (2027.5월 오픈): 사내 공유문서 (하이미, 하이드라이브) 연동 기반 질의응답 (하이미 통합검색창과 추후 연동), 출처 문서 접속 링크, 문서 요약 정보 제공
- **3단계** (2027.10월 오픈): 사용자가 업로드한 문서 및 실시간 웹 검색 기반 질의응답, 요약, 번역, 보고서 생성 기능 제공

### 시스템 구축 방향
- 공통: 내부망에서 접속 가능한 AI Portal(가칭) 링크 접속하여, AI서치(가칭) 서비스 접속
- 2단계 구축 시, 하이미 통합검색 창 하단에 검색(요약) 결과가 나오도록 API 연동 필요 
- 1단계의 경우, GPU 구매 이슈로 인해 기구축된 분석계(A100) 자원 활용하여 개발(및 운영?) 필요

### 주요 사용자
- 당사 임직원
